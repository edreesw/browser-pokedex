# browser-pokedex

A Pokédex in your browser, built using HTML, CSS, and Javascript. Data is dynamically retrieved using the publicly available PokéAPI (https://pokeapi.co/).

Searches for Pokémon by their name or id and displays some basic information about them on the page. 

# Known Issues

PokéAPI returns Pokémon cries (which are triggered with the "Play Sound" button) in the OGG format, which isn't currently supported by Apple/Safari based browsers.
